UPLOAD INSTRUCTIONS
1. Rename the GitHub repository to living-in-the-delta if you have not already.
2. Open the repository and upload all six files above into the root/main area.
3. Replace the existing files when GitHub asks.
4. Commit the changes.
5. Go to Settings > Pages and confirm:
   Source: Deploy from a branch
   Branch: main
   Folder: / (root)
6. Wait 1–3 minutes.
7. Hard-refresh with Command + Shift + R.
